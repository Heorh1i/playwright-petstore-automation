# Playwright API Automation Framework 🚀

This is an educational project focused on API test automation using Playwright and TypeScript. The framework implements modern testing patterns and best practices for building scalable and reliable automated tests.

## 🛠 Tech Stack
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Test Runner:** [Playwright Test](https://playwright.dev/)
* **Validation:** [AJV](https://ajv.js.org/) (JSON Schema Validation)
* **Data Generation:** [Faker.js](https://fakerjs.dev/)
* **Utilities:** Dotenv, Genson-js

## 🌟 Key Features
* **Automated Schema Validation:** Uses AJV and Genson-js to automatically generate and validate API response structures.
* **Custom Matchers:** Extended Playwright `expect` for cleaner and more readable test assertions.
* **Dynamic Data Generation:** Integrated with `@faker-js/faker` to ensure independent and randomized test data.
* **Environment Management:** Secured configuration handling using `.env` files.
* **Asynchronous Flow:** Clean implementation of `async/await` patterns for stable network interactions.
